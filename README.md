## Apa Itu Yaping 
  
Yaping adalah platform jejaring sosial berbasis web (gaya Facebook 2008) yang dibuat dengan HTML, CSS, dan JavaScript murni — tanpa framework, tanpa build step. Backend menggunakan **Supabase** untuk database, autentikasi, dan realtime. 
  
 --- 
  
 ## System Yaping
  
| Layer | Technology | Description |
|--------|------------|-------------|
| 🎨 Frontend | **HTML5, CSS3, Vanilla JavaScript** | Lightweight and responsive client-side interface without external frameworks |
| 🗄 Database | **Supabase (PostgreSQL)** via REST API | Cloud-hosted relational database with scalable backend services |
| 💾 Local Cache | **Browser localStorage** | Stores temporary local data for faster loading and offline-friendly behavior |
| 🔐 Authentication | **Supabase Auth** | Secure email & password authentication system |
| ⚡ Realtime System | **Supabase Realtime WebSocket** | Live synchronization for messages, updates, and collaborative features |
| 🔗 Peer-to-Peer | **PeerJS v1.5.4 (CDN)** | Direct P2P communication layer for realtime connections and media transfer |
